## 如果你用開chromium broswer (i.e. chrome, opera, edge)
1. 安裝一個叫「Get cookies.txt LOCALLY」的extension
假設你要拿到登入bilibili的cookie
2. 打開bilibili網站，確保你已經登入左　且打開左自動登入
3. 在bilibili那個tab執行Get cookies.txt LOCALLY
4. 在Get cookies.txt LOCALLY裡按Export
順利的話會下載了一個txt，這個就是需要的cookie。我的話會放在cookie folder(這個folder)裡
